# 🌐 Decentralized Java Network 🚀

⚡ A Java-based decentralized communication system inspired by torrent-style networking.  
🔗 Clients connect to known nodes to discover active servers and peers.  
🌍 No central authority — fully distributed node discovery system.  
🤝 Users can find and connect directly with other peers dynamically.  
🛡️ Built for scalability, resilience, and efficient distributed communication.
